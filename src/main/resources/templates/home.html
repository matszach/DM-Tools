Welcome to LK's DM tools

I created this web application to practice the basics of Spring framework.

It is supposed to serve as a tool for Dungeon Masters (hence the name) running classic fantasy Role Playing Games (eg. Dungeons&Dragons)

Following tools were used in creating this application:
- Java 11
- HTML/CSS
- Thymeleaf
- Spring MVC framework
- Spring Boot Initializr
- Maven
- Intellij IDEA IDE
- Visual Studio Code editor
